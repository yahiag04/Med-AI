# PROJECT REQUIREMENTS 

## Objective
the goal is to use the CNN model created (that detects pneumonia from xrays) in a web app that will allow doctors to have a overview of the results of different x rays. It's a web-app with authentication, a hub that allows to add an xray, create patient info and associate xray to the patient.
the model will be used to create a order of urgency based on the prediction made by the model (if pneumonia is detected then it should be placed in first position (rush))

## In Scope
- xray submission (its a pic)
- auth
- save the data in a database (can be sqlite for now)
- good, minimal and modern UI (keep light colors)


## Constraints
- Python + FASTAPI
- sql lite for now
- use html, bootstrap

## Rules
- analize well the code structure and see how everything is done (see how the model works)
